Commands

Command Description Examples Shortcuts
$spark [crystals/tickets/10parts] [set/add/subtract] [number] Allows you to manage your spark profile. $spark crystals set 420
$spark tix +1
$spark xtals minus 3000		 Crystals <=> xtals
Tickets <=> tix
10parts <=> 10tix or 10p
add <=> +
subtract <=> - or minus
$spark [create/reset/delete/profile] [nickname/username/@mention] Allows you to create, reset, delete, or display your spark profile. Also allows you to search for and display someone else's spark profile. $spark profile
$spark create
$spark profile Cryo		 $spark profile <=> $spark
$time Displays the current time in Japan.
$translate [text] to [language] Translates the given text to the given language. If language is not given, output text will be in English. Supports most languages supported by Google Translate. $translate 雪花 to Japanese
$translate あなたのお母さんゲイ
$help Direct messages the list of commands to you (which is this website).
=greeting [message/background/On/Off] Allows a moderator or higher role to customize and turn on/off the welcome message for new members. (Message refers to the message sent along with the image and not the one in the image itself). =greeting message Welcome to the server, @member!
=greeting background [img link]
=greeting on

Features

Custom spark profile in the theme of Granblue Fantasy, complete with a progress bar and spark percentage.

Customizable join message for new members.
