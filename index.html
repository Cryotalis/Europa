Commands

Command Description Examples Shortcuts
$spark [crystals/tickets/10parts] [set/+/-] [number] Allows you to manage your spark profile. $spark crystals set 420
$spark tix +1
$spark xtals minus 3000		 Crystals ⇄ xtals
Tickets ⇄ tix
10parts ⇄ 10tix or 10p
$spark [create/reset/delete/profile] [user] Allows you to create, reset, delete, or display your spark profile. Also allows you to search for and display someone else's spark profile. $spark profile
$spark create
$spark profile Cryo		 $spark profile ⇄ $spark
$time Displays the current time in Japan.
$translate [text] to [language] Translates the given text to the given language. If language is not given, output text will be in English. Supports most languages supported by Google Translate. $translate 雪花 to Japanese
$translate エウロペ
$help Direct messages the list of commands to you (which is this website).
=greeting [msg/bg/channel/display/on/off] Allows an Administrator to customize and turn on/off the welcome message for new members. Use "@member" in your message if you'd like the new member to be pinged. BG accepts both image links and image uploads, set BG to "default" for the default background. Channel accepts both channel ID and channel mentions. =greeting msg Welcome to the server, @member!
=greeting bg [image link]
=greeting on
=greeting channel #welcome		 $greeting on/off ⇄ $greeting

Features

Automatically updates spark counters in nicknames.
Opt-in by using this format: (xxx/300).

Custom spark profile in the theme of Granblue Fantasy,
complete with a progress bar and spark percentage.

Customizable join message for new members.
