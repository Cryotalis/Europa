Commands

General Commands

Command Description Examples Shortcuts
$spark [set/+/-] [crystals/tickets/10parts] [number] Allows you to manage your spark profile. You may set all 3 resources with one command by using $spark set X crystals, Y tickets, Z 10parts. If you prefer the legacy look of the spark profile, use $sparkle instead. $spark crystals set 420
$spark tix +1
$spark xtals minus 3000		 Crystals ⇄ xtals
Tickets ⇄ tix
10parts ⇄ 10tix or 10p
$spark [create/reset/delete/profile] [user] Allows you to create, reset, delete, or display your spark profile. Also allows you to search for and display someone else's spark profile. If you prefer the legacy look of the spark profile, use $sparkle instead. $spark profile
$spark create
$spark profile Cryo		 $spark profile ⇄ $spark
$spark background [link] Sets a custom background for your spark profile. Using $spark background without a link will reset your background to the default background. You can also set your background by uploading an image along with the command. $spark background [img link]
$crew [name/id] [CrewName/CrewID] Searches for a crew given crew name or crew ID. $crew name Mint
$crew id 1655580
$player [name/id] [PlayerName/PlayerID] Searches for a player given player name or player ID. $player name Europa
$player id 21213996
$events Displays current and upcoming events. Use $event if you prefer the text only version of the command. $events
$translate [text] to [language] Translates the given text to the given language. If language is not given, output text will be in English. Supports most languages supported by Google Translate. $translate 雪花 to Japanese
$translate エウロペ
$ask Ask Europa a yes/no question. $ask do I yeet rolls?
$choose Ask Europa to choose between 2 or more things. $choose homework, gbf
$time Displays the current time in Japan. $time
$feedback [feedback] Send feedback for Europa to me. $feedback Europa is awesome!
$bugreport [bug] Report a bug to me. $bugreport $spark isn't working.
$help Displays a link to this website and a link to Europa's support server. $help
=greeting [msg/bg/channel/display/on/off] Allows an Administrator to customize and turn on/off the welcome message for new members. Use @member in your message if you'd like the new member to be pinged. BG accepts both image links and image uploads. Set BG to default for the default background. Channel accepts both channel ID and channel mentions. Greeting is off by default. =greeting msg Welcome to the server, @member!
=greeting bg [image link]
=greeting on
=greeting channel #welcome		 $greeting on/off ⇄ $greeting

Command Description Examples Shortcuts

Features

Customized Spark Profile

Granblue Fantasy themed, includes a progress bar with spark percentage.

You'll get a congratulatory message when you reach 100% spark progress, and the progress bar can overflow until 200%.




Customizable Join Message

Customize the background and join message, change the channel the message is sent in, and use the display option to see what the message looks like before you enable it.









Auto-Updating Spark Counters

Make the spark counter in your nickname automatically update as you key in your spark funds.

Accepted formats: (000/300) & 00.00%

Europa's role must be in a higher position than the roles of those whose nicknames she is trying to update in order for this to work.

This also does not work for the server owner.










Player Search

Search for a player given ID or player name. Displays a screenshot of the player's profile, and neatly presents the player's main support summons, misc. support summons, and star character along with the star character message. Support summons have their uncap level displayed via stars.












Crew Search

Search for a crew given ID or crew name. Displays the crew's ranking in the previous Unite and Fight as well as name history.
