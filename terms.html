Privacy and Data Usage Policy

I don't collect any data other than what is necessary, and of course, I don't sell your data. Here are all the details regarding data collection:

  • Europa's database is partially available at Europa Database. Anyone may view and make use of the information in the database. The private portion of the database (it is a separate spreadsheet) contains server settings information for the greeting and role commands, spark profile information, and announcement/reminders information. Obviously, whatever you submit to the aforementioned commands will be stored in the database so Europa can call upon the information when you run commands.
  • Previously, Europa stored translated text in the private database, as the old translate command relied upon Google Spreadsheet formulas. It now uses Google's Cloud Translate API, so all translated text in the database has been deleted.
