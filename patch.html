What's New? (Update V2)

New Role System

Your typical role system. Administrators can add/remove roles to/from a list of roles with =roles. Regular users can view
the list of roles with $listroles, add roles to themself with $addrole, and remove roles from themself with $removerole.
Additionally, Administrators can organize roles into groups by adding them to a category.

New Events Commands

$event displays a plain text list of current and upcoming events, while $events displays a list of current and upcoming events with images.



Set Custom Spark Backgrounds

You can now set a custom background for your spark profile.

New "Fun" Commands

$ask Europa a question, or ask Europa to $choose between two or more things.


For information on how to use any of these commands, please see the command list.

Changes and Improvements

$player and $crew
  • The "Maximum Capacity Reached" error message should appear less often, and capacity should now be freed faster.
  • Added page number to player search when searching by name.
  • Added error message for when GBF is undergoing maintenance.
  • Player ID is now included in the screenshot in place of class name and class level.
  • Crew ID is now placed underneath Crew name in the embedded message, and is also placed at the top of the screenshot.
  • Improved speed and stability.
Miscellaneous
  • $help command now sends the website link to the channel the command was called in instead of sending a DM.
  • Leave and ban messages have been added to the $greeting system.
  • Replaced the FAQ page with the Upcoming page. The Upcoming page will contain information about future plans with
    regards to Europa.

Bug Fixes

  • Feedback and Bugreport commands now send an error message when the bug report or feedback is left blank.
  • Player search by name no longer requires the Manage Messages permission to work correctly. It will continue running
    without clearing all reactions if Europa is not given the Manage Messages permission.
  • Fixed bug where spark profile names were not being centered when using spark profile search.
  • You can no longer set your spark count to a negative number.
