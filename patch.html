What's New? (Update V3 | Released on May 27th, 2021)

Database Migration

The way greeting settings and role settings are stored were changed in this update. I've done my best to migrate everyone's data over properly, but if you notice that you're now missing your greeting data or roles data from before, feel free to reach out to me to get your data migrated properly.

$summons

Generates an image that shows all your support summons, your player name, and your player ID.

$roll

Gacha simulator that automatically stays up to date with the current banner. (The one below is from 5/23/2021)

=announce and =announcements

Allows you to have Europa make announcements at certain times as well as repeat those announcements at a specified interval. This might be useful for announcing striketimes or dailies etc.

$remind and $reminders

It's the announcements system except you get reminders private messaged to you instead :)

$mock and $drunk

cpnVErts TEXU INtO mOcLinG sPpngebOb teXT or dRUnk texu.

$question

Generates a random conversation question.

$role

Gives you the role you specified if you didn't already have it, and removes it from you if you already did have it.

=prefix and =modprefix

Allows you to set a custom prefix or mod prefix for your server.

=disable and =enable

Allows you to blacklist or whitelist commands across the server or in certain channels. Whitelisting is slightly more advanced, see the corresponding Advanced Usage Tips and Information section for more info.
Also, =disabled shows you which commands are disabled on your server.

@Europa

Ping Europa and she'll tell you what the current prefix and mod prefix is on your server.


For information on how to use any of these commands, please see the command list.

Changes and Improvements

Website Redesign
  • Shortcuts column was removed because it was taking up unnecessary space
  • Shortcuts and other additional usage tips are now placed in the description in a collapsible
  • The website should be a lot more useable on more screen sizes
$spark
  • Added new progress bars for those who have reached over 200% spark progress.
  • Europa will now congratulate you if you saved up more than one spark whereas previously she didn't say anything.
  • $spark create has been removed, using any $spark command will now automatically create a profile for you if you didn't already have one.
  • Nickname updater now updates your nickname across ALL servers you and Europa share.
  • $spark can now accept a larger variety of formats. All previous formats still work.
  • You can now search for another user's spark profile by simply using $spark [name]
  • You can now use a bookmarklet to copy a spark command to your clipboard with all your resources filled in (See the corresponding Advanced Usage Tips and Information for more info).
  • $spark background will now reject your background if it is invalid
  • $sparkle output will have a format that takes into account whether you're on PC or Mobile, although this is still a work in progress and is not perfect.
$player and $crew
  • $player now includes all support summons, and has an improved formatting for the star character as well.
  • $player and $crew commands should now be more stable and reliable than before, although this is something that is currently still being worked on.
=greeting
  • Now has an Auto-Role feature (automatically assigns roles to members who join the server)
  • Join message, leave message, ban message, autorole, and join image can now all be turned on or off individually.
  • You can now customize the leave message and ban message in addition to the join message.
  • =greeting background will now reject your background if it is invalid
Role System
  • Roles now keep up with name changes in the role list.
  • $listroles now shows a mention of the roles on the roles list so that roles are easier to find (this does not actually ping members of your server).
$events
  • Improved output of $events and $event, and improved reliability of $events.
  • $events will now show a random banner image for the current event if there are multiple banner images for that event.
  • $events and $event now show the elemental advantage for any event with an elemental advantage (GW, Proving Grounds, etc).
Miscellaneous
  • Commands that allow you to list multiple things ($choose, $addrole, $removerole, etc) now allow you to separate your list with other characters such as slashes or periods among others.
  • $time now allows you to convert a date/time from one time zone to another.
  • Added various shortcuts for various commands.

Bug Fixes

  • $spark and the greeting system will no longer fail if you had set an invalid background (it shouldn't be possible for you to set an invalid background anymore anyway)
  • $events and $event now show the correct element advantage for all events with an element advantage (GW, Proving Grounds, etc)
  • I fixed a lot of other stuff but honestly I don't remember the rest and digging them up would be too much of a headache so, "A lot of other bug fixes" :)
