What's New? (Update V3)

Gacha Command

Not yet available

Automatically Generate Support Summons

Generate a card with your support summons automatically, so you don't have to visit websites and manually input your support summons when you want to recruit friends.

Custom Prefixes

Set a custom prefix for regular commands and mod commands for your server.

Reminders and Announcements

Set reminders and announcements to be sent at a certain time, and then optionally repeat at a certain time interval. Useful for announcing striketimes every day.

Disable commands

Disable commands that you don't want your server to use. Additionally allows you to only disable or enable commands in certain channels.

Miscellaneous Additions

  • $addrole and $removerole have been replaced by $role. If you didn't have the role, you get the role; if you already had the role, you lose the role.
  • You can now choose to enable server leave messages and ban messages for the greeting system.
  • New $mock command for generating mocking spongebob text

For information on how to use any of these commands, please see the command list.

Changes and Improvements

Website Redesign
  • The website has been reworked to be more useable on more screen sizes.
$spark
  • Introduced a new blue progress bar for those who have over 200% spark progress.
  • Added "plus" as an acceptable alternative to "add" or "+" and "bg" as an acceptable alternative to "background".
  • Improved search algorithm for $spark profile search.
  • Added congratulatory messages for if the user saved up more than 1 spark.
  • Removed $spark create and the profile creation process. The command now works right off the bat, so there are no interruptions.
  • Invalid background images (images that Europa can't access) for $spark background will now be rejected, and you will be warned
    if you have an invalid background.
  • Nickname auto-updater now updates nicknames across all servers instead of just the server you're on.
Miscellaneous
  • Number of users has been replaced by number of spark profiles in Europa's rotating status.
  • $events and $event now show the elemental advantage for Proving Grounds.
  • You can now ping Europa if you forgot what the prefix and modprefix for the server are.
  • Updated $sparkle to match $spark in terms of functionality, improved formatting for users on PC at the cost of mobile users PC master race.

Bug Fixes

  • $events and $event will no longer display the incorrect element if Proving Grounds is a current or upcoming event.
  • $spark will now display your spark profile instead of failing to output anything when you have an invalid spark background.
