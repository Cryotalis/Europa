The Slash Command Update (August 2022)

  • The current command documentation may be insufficient. And I apologize for that. If you have any questions, please visit the support server.
  • This website has received a redesign, and several new pages have been added.
  • Support for message commands (commands that use $ or =) has ended. Due to changes made by Discord, only slash commands will be supported going forward. If slash commands do not work on your server, please click the "Add To Server" button. It will prompt you to enable slash commands. You don't need to kick and re-invite Europa to enable slash commands.
  • Two new commands have been added:
    • banner - shows you the duration, featured characters, and rate ups on the current banner.
    • translate text - translates a message to your own language. Run this command by right clicking on a message, hovering over Apps, and clicking "translate text"
  • As a result of switching to slash commands, several commands were improved because they were rewritten or refactored:
    • player and crew commands should now work again. I am still investigating issues with memory that cause them to sometimes stop working.
    • player generates the player image noticeably faster than before, because images are now loaded in parallel rather than in sequence.
    • player name search is now significantly faster, because it now uses an API rather than visiting a website.
    • crew name search is also significantly faster now for the same reason as above.
    • events generates the events image faster now, as it now loads the images in parallel rather than in sequence.
    • events now shows a Discord timestamp for when the events end for the embedded message version.
    • spark and greeting will now automatically find the direct imgur image link if you provide an imgur link, whereas previously Europa would fail to grab the image unless you provided the direct link.
    • spark now tells you how many rolls you gained or lost when you update your spark funds.
    • translate now translates to your language (the language you have set through Discord) by default.
  • Several commands have been merged with others or removed:
    • event has been merged with events, and is now accessible through a slash command option.
    • summons has been merged with player, and is now accessible through a slash command option.
    • addrole and removerole have been removed in favor of role.
    • roles (the mod command) has been renamed to rolelist to avoid confusion with role.
  • The feedback and bugreport commands have been removed. Going forward, feedback and bug reports for Europa should be sent to the Europa Support Server instead. It's easier for me to respond to reports and feedback this way as well.
  • The custom prefix and blacklist/whitelist commands features have been removed. Since Europa has switched to slash commands, the only prefix now is /, and slash commands can be whitelisted and blacklisted through Discord settings (Server Settings > Integrations > Europa > Manage).
  • The announcements and reminders commands are currently unavailable. I am still working on bringing them over to Europa V4, but I just wanted to get this update out earlier so people have time to switch to slash commands.

Got a concern, suggestion, or request? Join the Europa Support Server.
