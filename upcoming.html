Europa

Discord Bot for Granblue Fantasy. Made by ❄Cryo❄#4704.


What's Next? (Future Plans)

All of these are simply plans and ideas. While most of these ideas will probably be implemented, none of them are guaranteed to be added.

Commands

  • Some kind of gacha command, similar to AcePictureBot's.
  • A reminder command, to remind you about striketimes or whatever you want.
  • A sticker command, for using emotes. (I'll have to go get permission from people and I'm not very sure where to start)
  • A "friends" command, for generating an image with support summons, so you can find friends :o.
    • I've noticed people using the screenshot generated by $player to recruit friends, this might make things easier.
  • A guides command, for pulling up useful guides for Granblue Fantasy.
  • A news command, for checking the most recent news. (News from the offical GBF news page probably)
  • A character/summon search commmand, which can display information about a character or a summon. (Maybe even include weapons?)
  • Ability to set custom prefixes through a =prefix command.

Changes and Improvements

  • Further improve this website and improve existing commands.
  • Add shortcuts for commands where possible
    • $spark background => $spark bg
    • $bugreport => $bug
    • $spark set X crystals, Y tickets, Z 10parts => $spark set X, Y, Z
  • Make the nickname auto-updater change all nicknames across all servers, if the nicknames are the same and if this is possible.
  • A blue colored progress bar if someone has over 200% spark progress
  • Merge $addrole and $removerole into one command, $role. This command would remove the role if you already have it, and add the role if you didn't already have it.
  • Change Europa's status to show number of spark profiles in the database rather than the total number of users between all guilds Europa is in.
  • Improve error messages for incorrect usage of commands.
  • Allow Administrators to switch Europa to use another language, such as Japanese or Chinese.
    • This one's a bit ambitious and will probably not happen unless I can get people to translate for me.

Open Source!

Europa will go open source after update V3, which should happen some time in Fall 2020 or Winter 2020. This will take some effort, but I believe
this will be quite beneficial for others looking to create Granblue Fantasy Discord bots or just Discord bots in general. If I end up stopping
development of Europa, I want to make sure others can still host her on their own and make changes as they see fit. I will have to decide whether
I want to allow others to be able to contribute to the offical build.

Side note: Because her database is actually a Google Spreadsheet, it may be a little difficult for others to host her on their own.
