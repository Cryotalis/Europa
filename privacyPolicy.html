Privacy and Data Useage Policy

I only store data that is necessary for Europa to function properly. All user data is kept private, and in the event of a breach, I would do my best to notify all users as soon as possible. Virtually none of the data I collect through Europa would be data that anyone can do anything with, but in the event that you entered in some kind of sensitive information for any Europa command (or if you just don't want your data to remain in my database), you may contact me to have it deleted. I do not give out your data to anyone and I do not collect any data other than what is listed below.

Most probably do not know that I use a Google Spreadsheet as a database. Here is a link to a blank copy of Europa's database: Europa Database Template

I will now outline below all the data that I store and provide reasons as to why I need to store this data:

Intro

Nothing important is stored here. I include an intro page on almost any spreadsheet I make.

Servers

Guild Name and Guild ID are stored so that Europa can identify your guild (server) when you use commands. Greeting, Roles, Prefix, Mod Prefix, and Commands (which commands are disabled) settings are all tied to your server.
Guild (Server) Name is stored so that Europa may fall back on it if she is unable to identify your guild (server) by the ID. When you kick Europa from your server, your server information will be kept in case you want her back. If for whatever reason you'd like me to delete this data from the database, feel free to contact me.

Spark

User Tag and User ID are stored so that Europa can identify you when you use the $spark command. Crystals, Tickets, 10 Parts, Percent, Rolls, Background are all tied to your ID and Tag.
User Tag is stored so that Europa may fall back on it if she is unable to identify you by your ID. If you'd like to delete all data on your spark profile, you can run $spark delete.

Translate

This tab is necessary for the $translate command to work, as I use Google Spreadsheet formulas to make it work. The contents of this sheet are periodically deleted, but this does mean that I can see the kinds of things that you are using the translate command for. Even if the contents were deleted right after command execution, I can still see what was once there through Google Spreadsheet's edit history, so I've opted to instead just automatically delete the contents every month or so.

Data

This tab stores data on all the weapons and summons in the game for Europa to reference for the $roll command. No user data is stored here.

Announcements

This tab stores all the announcements and reminders users create. Europa will restart herself periodically, and she can sometimes crash due to errors. The announcements and reminders are stored here so that she can continue sending them out and keeping track of them even if she needs to restart.
